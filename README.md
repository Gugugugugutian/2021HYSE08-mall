# 2021HYSE08-MALL

弘毅软工第8小组线上商城实践项目仓库。

## 1. 项目介绍

本项目为2024年春学期“软件工程”课程大作业，由2021级软件工程第8组开发。
本项目为线上商城实践项目，旨在通过此项目学习有关电商系统的设计、实现、维护、运营等知识。
