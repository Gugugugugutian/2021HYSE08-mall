## 用户接口

by 顾田

| 接口名            | 方式   | 参数                 | 说明               |
|----------------|------|--------------------|------------------|
| /user          | GET  | 无                  | 测试数据库连接          |
| /user/login    | POST | username, password | 登录，保存用户信息到cookie |
| /user/register | POST | username, password | 注册               |
| /user/logout   | POST | 无                  | 登出，清除cookie      |

### 关于登录

用户登录成功后，会保存用户信息到cookie中，cookie中保存了用户名、密码、用户id、用户权限等。

### 关于登出

用户登出时，会清除cookie中的用户信息。
